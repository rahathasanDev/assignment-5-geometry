What is DOM and what is the purpose of DOM?

Ans: The Document Object Model (DOM) is a cross-platform and language-neutral programming interface for accessing and manipulating documents. It represents a document as a tree-like structure of objects. The DOM defines the objects that comprise the document and their properties and methods. This allows programs and scripts to dynamically access and update the content, structure, and style of a document. The DOM is used by web browsers to render web pages. It is also used by JavaScript and other scripting languages to dynamically interact with web pages.

How will you select HTML elements using DOM? Name the DOM methods.?

Ans: DOM can select any HTML element and also change it's value . There are 5 methods in Dom for select HTML element

  1. getElementById(): This method takes an ID as its argument and returns a reference to the element with that ID.
  2. getElementsByTagName(): This method takes a tag name as its argument and returns a collection of all elements with that tag name.
  3. getElementsByClassName(): This method takes a class name as its argument and returns a collection of all elements with that class name.
  4. querySelector(): This method takes a CSS selector as its argument and returns a reference to the first element that matches the selector.
  5. querySelectorAll(): This method takes a CSS selector as its argument and returns a collection of all elements that match the selector.

What is event bubble?

Ans: Event bubbling is a concept in the DOM (Document Object Model) that describes how events propagate through a document. When an event occurs on an element, it is first handled by that element. If the element has no event handler for the event, the event is then propagated to the element's parent. This process continues until the event reaches the top of the document tree, or until an element with an event handler for the event is encountered.
